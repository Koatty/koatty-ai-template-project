# koatty-ai-template-project

Koatty 项目脚手架模板，用于 `koatty new <name>` 创建新项目。

## 目录结构

- `default/` — 默认项目模板
  - 入口 `src/App.ts`、配置文件、示例 Controller/Service
  - 空目录占位：model、dto、middleware、plugin、aspect

## 使用

由 [koatty_cli](https://github.com/Koatty/koatty-ai) 的 `new` 命令自动拉取，用户也可通过 `koatty template update` 更新到最新版本。
